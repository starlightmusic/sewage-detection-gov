# Product Usage Instructions

This document provides instructions on how to use the Government Complaint Management System, both as a citizen and as an administrator. It also includes details on how to use the developer tools for testing.

---

## 1. For Citizens: Submitting a Complaint

This is the primary public-facing feature of the application.

1.  **Navigate** to the main website URL.
2.  You will see the **"Submit a Sewage Complaint"** form immediately.
3.  **Fill in the details:**
    *   **Location:** Be as specific as possible (e.g., street name, landmark, ward number).
    *   **Description:** Clearly describe the problem.
    *   **Contact Number:** Provide a valid phone number for potential follow-up.
4.  **Upload a Photo:**
    *   Click the "Upload Photo" box.
    *   Select a clear photo of the issue from your device's gallery or take a new one. A preview of the selected image will appear.
5.  **Submit:** Click the **"Submit Complaint"** button.
6.  **Confirmation:** A success message will appear with your unique **Complaint ID** (e.g., `#1001`). Please save this ID for your records, as it can be used for future tracking.

---

## 2. For Administrators: Managing Complaints

This is the secure portal for authorized municipal officers.

1.  **Login:**
    *   On the main website, click the **"Admin Login"** button in the top-right corner.
    *   Enter your assigned username and password and click "Login".
2.  **Dashboard Overview:**
    *   Upon successful login, you will be taken to the Admin Dashboard.
    *   At the top, you will see summary cards showing the total number of complaints by status (Total, Pending, In Process, Completed).
    *   The main table, the **"Complaint Registry,"** lists all complaints with their key details.
3.  **Managing the Complaint Lifecycle:**
    *   **Assigning a Complaint:**
        *   Find a complaint with the status "Pending".
        *   Click the **"Assign"** button in its row.
        *   In the modal that appears, select an officer from the dropdown list and click "Assign Officer".
        *   The complaint's status will automatically update to "In Process".
    *   **Completing a Complaint:**
        *   Once a complaint has been resolved, find it in the dashboard (its status will be "In Process").
        *   Click the **"Mark Complete"** button.
        *   In the modal that appears, you **must** upload a photo showing the resolved issue (the "after" photo).
        *   Click "Mark as Completed". The complaint's status will update to "Completed".

---

## 3. For Developers & Testers: Using Test Tools

To accelerate testing and development, a set of developer tools has been added.

### How to Use the Test Data Seeding Feature

This feature instantly fills the complaint submission form with valid sample data.

You can trigger this feature in three ways:

1.  **Floating Button (Recommended):**
    *   Navigate to the main complaint submission page.
    *   A small, blue floating button with a "ðŸ”§" icon will be visible at the bottom-right of the screen.
    *   Click this button to instantly populate the form fields.

2.  **Keyboard Shortcut:**
    *   While on the main page, press **`Ctrl + Shift + D`** (or **`Cmd + Shift + D`** on macOS).

3.  **URL Parameter:**
    *   Add `?seed=true` to the end of the website URL (e.g., `https://your-site.pages.dev/?seed=true`). The form will be auto-filled on page load.

**Note:** After seeding the data, you still need to manually upload a photo before you can submit the complaint.
